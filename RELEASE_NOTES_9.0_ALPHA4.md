# AIOS 9.0.0-alpha.4 — OLX Agent: профили, стратегия, конкурентная разведка

**Дата:** 2026-07-21 · **Тесты:** 667/667 зелёные · Python ≥ 3.10

Пятый выпуск линейки OLX Android Agent: управление профилем, слежение за
конкурентами, стратегический советник, бутстрап свежего сервера и обход
портфелей конкурентов.

## Главное

### Профиль и настройки OLX
- `ProfileParser` — поля профиля (имя/телефон/email/город/о себе) и
  состояния переключателей настроек; kv-зеркало `olx_profile_kv`.
- `ProfileEditor` — изменения по принципу guarded-action: dry-run складывает
  `_pending_*`, на устройство — только с `confirm=True`.
- CLI `aios olx profile|profile-edit`, REST `/olx/profile*`.

### Слежение за конкурентами от своих объявлений
- `derive_query` из заголовков, скоринг связи Jaccard+цена+город,
  таблица `olx_competitor_links`, подсчёт демпингующих конкурентов и
  ценовая позиция (ранг среди похожих).
- **Обход портфеля конкурента**: `parse_seller_ads()` разбирает блок
  «Другие объявления пользователя» на странице объявления (защита от ложных
  триггеров + исключение просматриваемого объявления);
  `observe_seller_ads()` сохраняет весь портфель как рыночные наблюдения и
  связывает только достаточно похожие позиции, идемпотентно.
- REST `/olx/competitive|refresh|seller-scan`, CLI
  `aios olx competitive|competitive-seller`.

### Стратегический советник
- По каждому своему объявлению: KEEP / EDIT_PRICE / EDIT_CONTENT / REPOST /
  PROMOTE с приоритетами 1–3 и объяснениями.
- `advise_new_listings()` — активные ниши без покрытия портфелем: целевая
  цена (медиана × 0.97) и стартовый заголовок из рыночных ключевых слов.
- REST `/olx/advisor?new=1`, CLI `aios olx advisor`.

### Бутстрап свежего сервера
- `aios olx bootstrap` — план/установка: apt → Python deps →
  platform-tools → ADBKeyBoard → SDK cmdline-tools → образ Android 34 →
  headless AVD `aios-olx` → настройка устройства. Обёртка
  `tools/olx_bootstrap.sh`.
- `aios olx doctor` / `GET /olx/doctor` — чеклист готовности с подсказками
  исправлений.

## Цифры
- 667 автотестов (+19), ~34 REST-роута OLX, 25 CLI-подкоманд.
- AutoWatch дополнен конкурентными связями и действиями советника.
- Философия guarded-action сохранена: ничего не меняется на устройстве
  без явного `confirm`.
