# AIOS Architecture

## Core Components

- **Orchestrator** — Central task execution engine
- **Constitution** — 67 articles governing behavior
- **Evolution Manager** — 7-stage proposal pipeline
- **Federation** — Multi-node coordination (v4.0+)
- **ML Scorer** — Intelligent planning (v4.0+)
- **Marketplace** — Capability sharing (v4.1+)

## Layers

1. **Persistence Layer** — SQLite
2. **Core Engine** — Orchestrator + Subsystems
3. **API Layer** — REST + MCP
4. **Intelligence Layer** — ML + Evolution + Federation
5. **Ecosystem** — Dashboard, SDK, CLI, Helm
