# AIOS Version 3.0 Roadmap

**Статус:** В активной разработке  
**Дата начала:** 2026-07-21

## Основные цели версии 3.0

### 1. Усиленная автономия
- 5 уровней автономии (AutonomyLevel)
- Автоматическая адаптация уровня на основе успеха/неудач
- Запись истории действий и триггеров

### 2. Расширенный Capability Engine
- Динамическое добавление/удаление возможностей
- Проверка совместимости возможностей
- Автоматическое предложение новых способностей

### 3. Улучшенный Planner
- Многоуровневое планирование
- Зависимости между шагами
- Валидация планов

### 4. Мониторинг и наблюдаемость
- Prometheus-метрики (`/metrics`)
- Скрипт `monitor.py`
- Расширенный health-check

### 5. Docker-first deployment
- Полноценная поддержка Docker и docker-compose
- Готовые production-конфигурации
