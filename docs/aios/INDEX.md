# AIOS — Объединённая документация (best-of-each)

Единое дерево, собранное из 4 срезов проекта AIOS. Для каждой темы выбран **канонический** документ (самый полный), альтернативные версии — в `variants/`.

## Срезы (источники)
- `slice1_native` — исходный каркас репозитория (`docs/core`, `docs/memory`, `docs/constitution` и т.д.)
- `slice1_constitution` — папка `constitution/` репозитория (книги/статьи конституции v2.1.1)
- `slice2_exec_v2_1_1` — манифесты исполняемого слоя v2.1.1 (`docs/executable_layer`)
- `slice3_v6` — спецификация v6 / v6.1 Global Federation (`docs/aios_v6`, из `AIOS.zip`)
- `slice4_archive_2026` — **активно развивающийся** архитектурный архив 2026 (`docs/aios_archive_2026`, из `AIOS_Archive_2026.zip`). Дополняется дальше — считать живым источником.

Итог: канонических документов = **399**, вариантов = **52**, удалено точных дублей = **0**.
